## Getting Started

To get a local copy up and running follow these simple steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/brianangulo/bruteforcerer.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or yarn 
   
    ```sh
   yarn
   ```

3. Run dev server
   ```sh
   npm start
   ```
   or yarn 
   
    ```sh
   yarn start
   ```
